Multikart

created: 23/05/2023

Multikart is an eCommerce mobile app with full integration with your existing Shopify website. It offers app developers a fast and expressive way to build native apps on both iOS and Android using a single codebase. This app comes with around 30+ screens and will work on both Android and iOS platforms. Multikart has some additional features like shimmer effect, multi-currency, multi-language, RTL support. This UI enables you to develop beautiful and feature-rich apps. You can add any part of the code. You like and apply it to your code. Our code is well organized with all folders, file names, class names variables and functions under 70 lines. This code is well named to make it easy to reuse and customize. This app has features like light and dark mode.

Update Theme Visual Studio :

  1. Download zip code from Codecanyon.
  2. Now Extract the Zip.
  3. the extract project in Visual Studio.
  4. Now Copy src folder and paste in your project
  5. Now go to downloaded multikart project go to package.json and copy all package name and paste in your project.

  8. Now run react native project using below command
    9. npx react-native run-android

  10. Now is time to Build Release apk
  11. If you have Duplicate resource error You need to add Below command nodemodules -> react-native -> react.gradle -> Add on line no 219 after doFirst

iOS:
  1. Create an IOS or Android click on Add app
  2. Select an iOS app in the firebase console.
  3. Write bundlid and APPID
  4. Download the Googleservice-info.plist file
  5. Replace the file in "ios/Runner/info.plist"
  6. Update "FirebaseApp.Configure()" in AppDelegate File
